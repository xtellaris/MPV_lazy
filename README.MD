## 简易包 mpv.net_CM

简易包是基于mpv.net（一个基于libmpv的前端程序）的中文修改版，更加易于上手。除了涵盖了懒人包的大部分选项外，还实现了一些额外的实用功能与基本的图形交互，更接近普通用户的使用方式。  
传送门 [mpv.net_CM](https://github.com/hooke007/mpv.net_CM)
